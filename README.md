# web-service-lzk
使用express搭建的简易物联网web服务器

需要配置node环境
需要配置mysql环境
  需要建立数据表 ，可参考代码

默认端口3000

浏览器访问:http://localhost:3000
api端口为 http://localhost:3000/api
  支持 RESTful,MQTT
